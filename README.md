## Available Operators

|Operator|Description | Positioning|
|---|---|---|
|+|Addition|prefix|
|-|Subtraction |prefix|
|*+*|Multiplication|prefix|
|/|Division|prefix|
|(|Bracket open|prefix|
|)|Bracket close|postfix|
|,|Coma|infix|
|^|Power|infix|
|!|Factorial|postfix|
|abs|Absolute|function|
|min|Minimum|function|
|sqrt|Square root|function|
|sin|Sin function|prefix|
|cos|Cos function|prefix|
